# idb
In early July 2020, Kuwait Source startup published the news of a project in the field of security and privacy policies on the official page of Aria Behroozian on Instagram A little later, Aria Behroozian announced more details about the project in a video posted on YouTube According to the developers and the development team of this project, this service is used to build and design databases so that users can design their database maps in a software development environment safely And now, in early 2021, this startup announced the official release of this project, which will be activated for public access from the beginning of the new month, ie on 1/2/2021.
